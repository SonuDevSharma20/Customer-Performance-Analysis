# Customer-Performance-Analysis
This project focuses on analyzing Customer Performance Analysis data using Python (Pandas) and visualizing key insights through an interactive Power BI dashboard. The main objective is to understand sales trends, product performance, and profitability across different categories, sub-categories, and regions.
